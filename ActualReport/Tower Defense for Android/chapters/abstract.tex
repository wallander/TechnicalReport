\section*{Abstract}

This thesis focuses on game development for Android, an operating system for mobile phones developed by Google. The market for applications for Android is growing rapidly and a large portion of these are games. The purpose of this thesis is to find out how to develop a competitive Tower Defense game on this expanding market.

Tower Defense is already an established game concept, there are several games of this type with different themes already on the market. In order to provide a solid foundation from which a new Tower Defense game can be developed, several popular games were examined thoroughly to find desirable features. In addition to this, interviews with people that play Tower Defense games regularly were conducted, making sure no important features were left out.

The research concluded that the game needed to be innovative to ensure that it would be competitive on the Android market. Android smartphones primarily provide two features that were utilized when developing Eskimo Tower Defense: the touchscreen and the accelerometer. The touchscreen provides a new method of interacting not yet available for computers. However, this alone was not enough to distinguish a Tower Defense game on the Android platform. Eskimo Tower Defense uses the accelerometer to give players additional ways to control the game, making them more physically involved in the gameplay. 

\section*{Sammanfattning}

Den här rapporten fokuserar på spelutveckling för Android, ett operativsystem för mobiltelefoner utvecklat av Google. Marknaden för applikationer till Android växer kraftigt och en stor del av dessa är spel. Syftet med det här arbetet är att utreda hur ett konkurrenskraftigt Tower Defense-spel kan utvecklas för den här växande marknaden.

Tower Defense är redan ett etablerat spelkoncept, det finns ett flertal spel av den typen med olika teman på marknaden. För att skapa en stabil grund för utveckling av ett Tower Defense-spel undersöktes flera populära Tower Defense-spel utförligt för att hitta önskvärda drag. Utöver detta genomfördes intervjuer med personer som spelar Tower Defense-spel regelbundet, för att tillse att inga viktiga funktioner utelämnades.

Undersökningar visade att spelet behöver vara innovativt för att garanterat vara konkurrenskraftigt på Android-marknaden. De funktioner som främst tillämpades under utvecklingen av Eskimo Tower Defense var pekskärmen och accelerometern. Pekskärmen möjliggör interaktion på ett sätt som inte är möjligt med datorer. Detta var dock inte tillräckligt för att urskilja ett Tower Defense spel på Androidplatformen. I Eskimo Tower Defense tillämpas accelerometersensorer för att ge användaren fler sätt att kontrollera spelet, vilket leder till en mer involverande spelupplevelse.
